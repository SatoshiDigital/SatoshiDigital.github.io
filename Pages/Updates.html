    Instaladores de Programas

    Após formatar seu computador ou instalar um novo você talvez procure meios mais rápidos do que instalar um por um todos os programas que você deseja. Neste artigo irei apresentar alguns meios mais fáceis de instalar seus programas tanto no Windows como no Linux(Baseado em Ubuntu).

    Ninite - Windows



    Ninte é um site onde você pode escolher entre vários programas e criar um instalador para os programas, de fato é uma plataforma muito útil, porém limitada em relação aos programas que você pode escolher.
    Aqui deixarei uma lista com alguns instaladores com vários programas.

    Ninite Basic
    - Chrome
    - LibreOffice
    - VLC
    - WinRAR
    - Java-86x e 64x
    - TeamViewer
    - Paint.net

    Ninite Pro
    - Chrome
    - Firefox
    - Foxit Reader
    - LibreOffice
    - CutePDF
    - VLC
    - 7zip
    - WinRAR
    - Java-86x e 64x
    - TeamViewer
    - Paint.net
    - Inkscape
    - Drive for Desktop

    Para executar o Ninite você só precisa baixar o arquivo .exe e executá-lo em seu computador.

    Arquivo Batch - Windows



    Já se você quiser instalar outros programas que não se possui no Ninite, podemos criar um pacote Batch com todos os programas que quiser. Irei disponibilizar um pacote com vários programas e logo mais postarei um tutorial completo de como criar seu instalador Batch.

    Update-Win-1.3.bat
    - Chrome
    - Drive for Desktop
    - Firefox
    - Adobe Acrobat Reader
    - WinRAR
    - Anydesk
    - 7Zip
    - LibreOffice

    Ele é um arquivo bat, ou seja um script executável, para executá-lo basta seguir os seguintes passos:
    1- Baixe ou Copie o arquivo "Update-win-1.3.bat" para o PC.
    2- Clique com o botão direito do mouse no arquivo.
    3- Clique em "Executar como Abministrador".
    4- Se o Windows não executar. clique em "Mais informações" em seguida em "Executar assim mesmo".
    5- Concorde com os termos de instalação dos programas.
    6- Aguarde até o fim da instalação.
    7- Clique “Enter” para sair da instalação.
    8- Após isto exclua o arquivo "Update-win-1.3.bat".

    Arquivo SH - Linux



    No Linux não temos um site para criar um pacote de instalação, irei disponibilizar um pacote com vários programas que eu criei e logo mais postarei um tutorial completo de como criar o seu próprio pacote para Linux.

    Update-1.0.sh
    - Chrome
    - WPS
    - LibreOffice

    O arquivo “Update-1.0.sh” tambem é um script executavel, basta seguir os passos para executá lo:
    1- Baixe o arquivo “Update-1.0.sh” para a "Área de Trabalho(Desktop)” do PC.
    2- Clique “Tecla de ação(Tecla do Windows)+Alt+T”, todas ao mesmo tempo para abrir o Terminal.
    3- Em seguida digite o comando “cd ./Desktop” para ir a "Área de Trabalho”.
    4- Digite o comando “ls” para ver os arquivos presentes na "Área de Trabalho”.
    5- Localize o arquivo “Update-1.0.sh”.
    6- Digite o comando “sh ./Update-1.0.sh” para executar o arquivo.
    7- Ele pedirá a senha do Usuário, caso não possua, fale com o administrador do PC.
    8- Concorde com os termos da instalação e siga os passos descritos nela.
